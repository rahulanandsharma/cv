Q4 a)

Camera Calibration

Usage

[P, K, R, t] = camcalib (Filename, Numpts)

Ex: [P, K, R, t] = camcalib (‘ImageDetails.tx’, 100)

Input is given in the form of file “ImageDetails.txt” which contains image to world point correspondences

Implementation:

 

If x is image point and X is world point

        x = P*X

Step 1: Normalization of the Points.

Step 2: Apply SVD (Singular Value Decomposition)

Step 3: Get Last singular vector as P

Step 4: Denormalize P

 

Decompose the camera projection matrix P into intrinsic               matrix K, rotation matrix R, and translation vector t.

 

P: 3x4 camera projection matrix
 
K: 3x3 intrinsic matrix
R: 3x3 rotation matrix
t: 3x1 translation vector
 
We can specify number of points to use for camera calibration.
